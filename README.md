# 🐍 [2026-01] 나만의 파이썬 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* [ 스마트 뷰티 컨설턴트 - 맞춤형 스킨케어 추천 서비스 ]
<br>\\


## 2. 시나리오 (5~10줄)
* 이 프로그램은 사용자 맞춤 스킨케어 제품을 추천해주는 나만의 뷰티 컨설턴트 시스템 입니다. 단순히 제품 추천 외에도 화장품에 관련한 다양한 기능을 추가하여 뷰티 백과사전 프로그램 처럼 구상하였습니다.

* 사용자가 본인의 피부 고민, 연령대, 찾고 싶은 화장품 카테고리를 입력하면 저장된 데이터베이스 내 사용자가 입력한 것에 해당하는 제품 목록을 출력합니다. 
또한, 사용자의 정보를 입력하면 프로그램에 스킨-세럼-크림을 한 번에 맞춤형으로 추천해주는 스킨케어 루틴을 짜주는 기능이 있습니다. 이때 예산 또한 고려하여 입력한 예산 내에서 구매 가능 한 제품들로 추천하는 기능도 함께 추가하려고 합니다.
재미요소를 위해 운세 기능 또한 넣을 것 입니다. 이를 통해 사용자가 제품 추천을 받는 것 외에도, 주기적으로 이 프로그램을 방문하게 되는 요소가 될 것 입니다.
성분 궁합 기능을 추가할 것 입니다. 궁합이 궁금한 성분 2개를 입력하면, 같이 사용했을 때 좋을 지 안 좋을지 프로그램이 판단해서 결과를 출력합니다.
제품 구매를 할 때 살 제품도 많고 가지고 있는 쿠폰도 많다면 어떻게 분배해서 구매해야 가장 효율적으로 구매가 가능한지 최저가 쿠폰 계산기를 통해 가지고 있는 쿠폰의 종류와, 구매할 제품의 금액을 입력하면 프로그램이 스스로 계산해서 알려주는 기능을 추가할 것 입니다.
화장품의 특성상 실제로 사용해본 사람들의 후기가 중요한데, 사용자들이 제품 후기를 작성할 수 있는 기능과 데이터베이스 내 새로운 상품들을 사용자들이 직접 추가할 수 있게 한다면 더 정확하고 사용하기 좋은 프로그램이 될 것 입니다.



<br>

## 🟦 [1차 과제: V1.0] 시나리오 기획
    
### **🤖 AI 파트너십 과정**

 1.  **내용 1 : 나의 아이디어를 제공 후 AI에게 코드 설계 요청, AI가 만들어준 코드를 기반으로 실제로 프로그램 실행 후 수정하고 싶은 부분 수정 요청

     * **프롬프트 요약:** "내가 만들고 싶은 프로그램에 대해 소개해줄게.(이후 프로그램 설명)", "카테고리 선택 기능을 넣고 싶어.", "최저가 계산기에서 어떤 상품을 어떤 쿠폰을 이용해서 구매해야 되는지 안뜨고 최종 가격만 떠."
     * **적용 내용:** 나는 내가 만들고 싶은 프로그램이 확실히 정해졌기 때문에, AI에게 나의 아이디어를 말하고 이를 설계해달라고 요청함. 또한, AI가 만든 코드를 이용해 프로그램을 실행해보고 사용했을 때 아쉽거나 더 추가하고 싶은 부분과 코드에 오류가 있는 부분을 수정해달라고 요청함.

      
  2. **내용 2 : 프로그램에 들어갈 기능 추천받기

     * **프롬프트 요약:** ""추가 할 기능이 있을까? 재밌는 기능이 있으면 좋겠어." "스마트 예산 필터링은 맞춤 루틴 생성 기능에 합쳐서 금액을 설정할 수 있게 하면 좋을 것 같아."

     * **적용 내용:** 내가 생각한 기능 외에도 AI가 추천해준 오늘의 운세, 최저가 계산기, 성분 궁합 시스템이 괜찮은 것 같아서 추가함. 제품 추천이라는 시스템 외에도 재미 요소와 실용적인 요소들이 들어간 시스템이 추가되어 프로그램이 더욱 더 개성있어짐. AI 추천 기능에 내 아이디어를 덧붙이거나, 내가 제시한 기존 기능에 AI의 의견을 합쳐 더 나은 프로그램을 만들었음.

## 🟩 [2차 과제: V1.0] 입출력 + 리스트 + 조건문 - 향후 작성 예정
### **✨2차 과제 내용:**
(예시)
  * 이번 과제는 용사 등록 및 정밀 진단 시스템 구현으로, 사용자로부터 데이터를 입력받아 리스트에 저장하고, 가중치 연산을 통해 결과값을 도출하는 시스템을 구축하는 것입니다. 특히 `if-elif` 조건문과 `and/or` 논리 연산자를 활용하여 상황에 맞는 등급과 특별 칭호를 부여하는 지능형 로직을 완성해야 합니다.
  * input()으로 용사의 이름, 체력, 마력, 공격력을 입력받아 리스트에 저장.
  * 가중치 공식을 적용하여 종합 전투력 산출: (체력 * 0.7) + (마력 * 0.5) + (공격력 * 1.8).
  * if-elif-else를 사용하여 S등급부터 F등급까지 5단계 등급 판정 로직 추가.
  * and 연산자를 활용하여 '전설의 대마법사' 등 특별 칭호 부여 기능 구현.
    
### **🤖 AI 파트너십 과정**
(예시) 
1. **내용 1: 길드 매니저 컨셉 및 로직 도출**
    * **프롬프트 요약:** "내가 길드 매니저가 되어 용사를 관리하는 프로그램을 만들 거야. 1학년 수준에서 리스트와 조건문을 활용해 등급을 매기는 재미있는 로직을 짜줘."
    * **적용 내용:** AI가 제안한 '가중치 전투력 산출' 방식과 'S등급이면서 마력이 높으면 대마법사 칭호 부여'라는 복합 조건 아이디어를 실제 코드에 반영함.

2. **내용 2: 효율적인 리스트 활용법 자문**
    * **프롬프트 요약:** "파이썬 기초 문법 단계(리스트 → 조건문 → 반복문 → 함수/딕셔너리)에 맞춰, '용사 길드 관리자' 컨셉의 프로그램이 어떻게 발전할 수 있을지 4단계 기능 확장 로드맵을 제안해줘."
    * **적용 내용:** 단순 입출력을 넘어, **'기초 정보 입력 및 리스트 저장(1차) → 전투 등급 판독기 구현(2차) → 무한 메뉴 시스템 구축(3차) → 딕셔너리를 활용한 다중 캐릭터 관리(4차)'**로 이어지는 논리적인 확장 로드맵을 확립함.
    
### **🛠️ Troubleshooting & 기술 회고:**
(예시) 
1. **문제 1: 숫자를 입력했는데 계산이 안 되는 현상 (TypeError)**
    * **원인:** `input()`으로 받은 데이터는 기본적으로 '글자'라서 숫자와 곱셈 연산이 불가능했음.
    * **해결:** `int()`와 `float()` 함수를 사용하여 입력값을 숫자로 바꾸는 '형변환' 마법을 부려 해결함.
2. **문제 2: 조건문 뒤에 자꾸 생기는 빨간 줄 (SyntaxError)**
    * **원인:** `if`와 `elif` 조건식 끝에 마법의 기호인 `:`(콜론)을 자꾸 빠뜨려서 발생함.
    * **해결:** 파이썬의 모든 조건문과 반복문 끝에는 항상 땡땡(`:`)이 들어가야 한다는 규칙을 학습하고 수정함.
